# 🏋️ 新手健身追踪器

基于 [MuscleWiki](https://musclewiki.com) 数据的新手健身训练追踪网站。

## 训练模式

**练三休一 · 4日循环**：背 → 胸 → 肩 → 休息

每天 5 个动作，每个动作配有：
- 🎬 MuscleWiki GIF 动画演示（正/侧两面 + 男女模特 + 肌肉发力热力图）
- ▶️ B站中文教学视频
- 📊 详细训练参数（热身组/正式组/组间休息/动作节奏）
- 📝 逐步动作要领
- ⚠️ 常见错误提示

## 使用方式

直接双击 `fitness-tracker.html`，浏览器打开即用。

- 自动识别今天是训练日还是休息日
- 展开动作卡片查看完整教学
- "跳过今天"一键顺延计划
- 底部设置可修改起始日期

## 数据来源

全部 18 个动作（含 3 个备选）均来自 [MuscleWiki](https://musclewiki.com)，每个动作链接到对应的 MuscleWiki 动画演示页。
