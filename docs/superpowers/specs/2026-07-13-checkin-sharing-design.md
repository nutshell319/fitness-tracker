# 打卡激励 & 分享系统 — 设计规格

> 日期：2026-07-13 | 状态：已确认 | 方案：A（渐进增强）

---

## 概述

在现有健身追踪器基础上叠加**打卡激励层**和**分享系统**，不改动现有勾选逻辑，新增模块全部独立。核心目标：让用户乐意打卡、乐意分享。

**约束**：单文件 HTML（index.html），零外部依赖，localStorage 存储，不改动现有功能。

---

## 模块一：打卡激励系统

### 1.1 每日打卡按钮

**位置**：Header 区域，日期下方，现有按钮组之前。

**交互逻辑**：

| 状态 | 按钮样式 | 行为 |
|------|----------|------|
| 未打卡、0 动作完成 | 🔥 今日打卡（灰色/禁用） | 不可点击，hover 提示"至少完成1个动作才能打卡" |
| 未打卡、≥1 动作完成 | 🔥 今日打卡（主色 #ff6b35 填充，脉冲动画） | 可点击，触发打卡 |
| 已打卡 | ✅ 今日已打卡（#22c55e 绿色） | 不可点击，显示已完成 |

**点击后流程**：
1. 记录当日打卡（localStorage key: `ft_checkin_YYYY-MM-DD`，值: timestamp）
2. 更新连续天数、检测是否达到里程碑
3. 触发庆祝动画
4. 按钮切换为已打卡状态
5. 如达到里程碑 → 额外弹出徽章获取动画
6. 动画结束后弹出分享日签引导（轻提示，不强制）

**周边调整**：
- "⏭ 跳过今天" 缩为次要小按钮（文字按钮，放打卡按钮右侧）
- "📝 自定义计划" 保持现有位置

### 1.2 打卡庆祝动画

**实现**：Canvas 覆盖层（固定定位，pointer-events: none）

**粒子爆炸**（~1.2s）：
- 50 个粒子从按钮中心飞溅
- 颜色：橙色 (#ff6b35) + 青色 (#4ecdc4) 混合
- 粒子行为：随机方向、重力下落、大小衰减、透明度从 1→0

**连续天数弹窗**（~1.5s）：
- 大字从 scale(0) → scale(1.2) → scale(1) 弹入
- 数字从 0 滚动到实际值（如 "🔥 X 天连续！"）
- 如果是中断后第一天：显示 "👊 重新出发！第一天"
- 如果达到里程碑：文字使用徽章对应颜色

**鼓励语**（~2s）：
- 从底部滑入，停留 1.5s 后滑出
- 随机选择（共 12 条预设），按场景分类：

| 场景 | 鼓励语示例 |
|------|-----------|
| 新开始（连续 1-2 天） | "每一个传奇都始于第一步，今天就是你的第一步" |
| 进行中（连续 3-6 天） | "坚持不是一时的爆发，而是日复一日的选择" |
| 小成（连续 7-13 天） | "一周的汗水不会白流，身体已经开始改变" |
| 习惯（连续 14-29 天） | "两周了！这不是坚持，这已经是你的生活方式" |
| 突破（连续 30+ 天） | "三十天的自律，换来一个全新的自己" |
| 里程碑通用 | "解锁新成就！每一次打卡都是对自己的承诺" |
| 中断回归 | "回来就好，中断不算失败，放弃才是" |

### 1.3 连续打卡追踪

**存储**：
- `ft_checkin_YYYY-MM-DD` → timestamp（打卡时间）
- `ft_longest_streak` → number（历史最长连续天数）

**计算逻辑**：
```
currentStreak：从今天往前数，遇到跳过日期停止，遇到未打卡日期停止
longestStreak：max(当前连续, 历史最长)
```

**UI 展示**：
- 热力图统计区第 4 个数字：🏆 最长纪录
- 连续天数旁边显示进度提示：
  - "还差 X 天解锁 💪 渐入佳境"
  - 达到里程碑时显示 "🎉 已解锁全部徽章！"

### 1.4 里程碑徽章

**7 级徽章**：

| 天数 | 图标 | 名称 | 颜色 | 描述 |
|------|------|------|------|------|
| 7 | 🔥 | 初露锋芒 | `#ff6b35` | 坚持一周，你已经超越了昨天的自己 |
| 14 | 💪 | 渐入佳境 | `#4ecdc4` | 两周不间断，身体的反馈正在路上 |
| 30 | ⚡ | 养成习惯 | `#f0c040` | 科学表明 30 天足以养成一个习惯 |
| 60 | 🏆 | 持之以恒 | `#a78bfa` | 两个月，健身已成为你的日常 |
| 100 | 🌟 | 百炼成钢 | `#f472b6` | 一百天的汗水，锻造成钢铁般的意志 |
| 180 | 👑 | 半载修行 | `#60a5fa` | 半年不间断，你已是人群中的 1% |
| 365 | 🎖️ | 全年无休 | `#e85d75` | 整整一年的自律，值得所有人的掌声 |

**存储**：`ft_badges` → `[{id, name, unlockedAt, streakAt}]`

**UI**：
- 位置：热力图格子下方、图例上方
- 样式：一排徽章图标，已获得亮色+弹跳动画，未获得灰色半透明
- 交互：已获得的徽章可点击，弹出详情（解锁日期+当时连续天数+描述）
- 解锁时：全屏弹出专属动画（徽章图标放大旋转 + 名称大字 + 描述），持续 2.5s

## 模块二：分享系统

### 2.1 每日打卡日签

**入口**：
- Header 打卡成功后自动弹出引导按钮
- 热力图统计区的 📸 日签 按钮
- 底部常驻栏分享入口

**卡片规格**：
- 比例：3:4 竖版（750×1000 px @ 2x = 375×500 CSS px 逻辑尺寸）
- 由 Canvas 渲染，下载为 PNG

**卡片布局**（从上到下）：

```
┌──────────────────────────────┐
│         🏋️ 肩日               │  ← 训练部位图标 + 名称
│                               │
│       2026年7月13日            │  ← 日期（大字）
│          星期日                │  ← 星期
│                               │
│      🔥 连续打卡 7 天          │  ← 连续天数（主色高亮）
│                               │
│   ┌─────────────────┐        │
│   │  宽肩撑起气场，   │        │  ← 激励语卡片（半透明背景）
│   │  饱满三角肌      │        │
│   └─────────────────┘        │
│                               │
│   完成 4/5 动作               │  ← 完成统计
│   ████████████░░░░  80%       │  ← 进度条
│                               │
│   • 哑铃侧平举  ✓             │  ← 完成的动作列表（最多5个）
│   • 杠铃站姿推举 ✓            │
│   • 蝴蝶机反向飞鸟 ✓          │
│   • 哑铃前平举 ✓             │
│   • 哑铃反向飞鸟 ✗            │
│                               │

│   FitnessTracker              │  ← 水印
└──────────────────────────────┘
```

**背景设计**：
- 底色：`#1e2127`（卡片色）
- 顶部装饰：训练部位对应颜色渐变微光（径向渐变，透明度 0.15）
- 底部装饰：一排训练部位颜色的小圆点

### 2.2 周报

**入口**：热力图统计区 📊 周报 按钮

**卡片规格**：16:9 横版（1200×675 px @ 2x）

**卡片布局**：

```
┌──────────────────────────────────────┐
│  📊 本周训练周报  7/7 - 7/13        │  ← 标题行
│                                      │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌─────┐│
│  │  4   │ │  1   │ │ 80%  │ │  🔥 ││  ← 四格数据卡
│  │训练天│ │休息天│ │完成率│ │ 7天 ││
│  └──────┘ └──────┘ └──────┘ └─────┘│
│                                      │
│  ┌─ 每日完成 ───────────────────┐   │
│  │ 一 二 三 四 五 六 日         │   │
│  │ ██ ██ ░░ ██ ░░ ░░ ██        │   │  ← 7天竖条（绿=练了，灰=没练/跳过）
│  │ ✓  ✓  ⏭  ✓  —  —  ✓        │   │
│  └──────────────────────────────┘   │
│                                      │
│  肌肉群分布                           │
│  🟠 胸 ████████░░ 40%                │
│  🔵 背 ██████░░░░ 30%                │
│  🟡 肩 ████░░░░░░ 20%                │
│  🟢 腹 ██░░░░░░░░ 10%                │
│                                      │
│  "这周练了4天，胸练得最多，继续保持！"  │  ← AI风格总结
│                                      │
│  FitnessTracker · 每周打卡           │
└──────────────────────────────────────┘
```

**数据来源**：
- 本周定义：周一至周日（`getWeekRange(today)`）
- 每日完成状态：检查当天是否有 `ft_checkin_` 记录 OR 有动作勾选记录 OR 不在跳过列表中
- 肌肉群分布：统计本周每天的训练部位

**一句话总结生成规则**（简单模板）：
```
训练天数 ≥ 5 → "这周全勤！你是真正的战士💪"
训练天数 ≥ 3 → "这周练了{X}天，{最多部位}练得最多，继续保持！"
训练天数 ≥ 1 → "这周练了{X}天，下周争取更多突破！"
训练天数 = 0 → "这周休息了一周，下周重新出发吧！"
```

### 2.3 月报

**与周报类似，差异点**：
- 数据范围：当月 1 号到最后一天（或到今天）
- 当月迷你热力图（复用热力图格子逻辑，缩小版，5×5 px 每格）
- 统计项：月训练天数/总天数、完成率、最长连续、最常练部位
- 右下角：下月目标（当前月数据 + 1 天的小目标）

## 模块三：UI 布局调整

### 3.1 Header 修改

- 打卡按钮插入到日期行下方、现有按钮组上方
- 跳过按钮样式降级为文字按钮

### 3.2 热力图区域调整

统计区变更：
```
旧：[训练天数] [跳过天数] [连续天数] [📤分享]
新：[训练天数] [跳过天数] [连续天数] [🏆最长纪录]
    [📤热力图] [📸日签] [📊周报] [📊月报] [📊年报]
```

徽章墙：热力图格子下方、图例上方

### 3.3 周期栏增强

每个日期格子右侧加小圆点：
- 绿色实心：当天已打卡
- 灰色空心：当天未打卡
- 红色：当天被跳过

### 3.4 底部常驻栏

现有撤销栏旁边加 📸 分享日签快捷入口。

### 3.5 新增 localStorage keys

| Key | 类型 | 说明 |
|-----|------|------|
| `ft_checkin_YYYY-MM-DD` | number | 打卡时间戳 |
| `ft_longest_streak` | number | 历史最长连续天数 |
| `ft_badges` | JSON array | 已解锁徽章列表 |

## 实施顺序

共 3 个阶段，每阶段完成后提交：

**Phase 1**：打卡按钮 + 庆祝动画 + 连续追踪 + 最长纪录
**Phase 2**：里程碑徽章系统 + 徽章墙 UI
**Phase 3**：分享系统（日签 → 周报 → 月报 → 年报）

---

## 设计约束

- 单文件 HTML，CSS/JS 全部内联
- 零外部依赖，Canvas API 完成所有图像生成
- 深色健身主题保持一致（配色变量复用现有 `:root`）
- 所有新增动画使用 CSS @keyframes + Canvas 粒子，不引入 JS 动画库
- localStorage keys 统一 `ft_` 前缀
- 移动端响应式（所有分享卡片在 375px 手机上也清晰可读）
- 代码注释中文，遵循项目现有规范
